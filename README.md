# Tic_Tac_Toe

Made with help from youtube project to learn javacript functions and especially arrow functions. 

Also learn css styling on different actions and variables in css. 

A simple tic tac toe game where the mission is to get either three x´s or o´s in a row both vertical, horizontal and diagonal. You and another player take turns clicking inside the board wich is splitted into 9 squares when you get three of the same in a row you win.
